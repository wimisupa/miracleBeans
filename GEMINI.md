# 프로젝트 지침 (miraclePoint)

## 서버 환경 (Server Environment)
이 프로젝트의 서버 포트 규칙은 다음과 같습니다. 프로젝트 작업 시 항상 이 포트 번호를 염두에 두고 작업하세요.

* **개발 서버:** 포트 `3000` 사용 (로컬 개발 환경, `npm run dev` 또는 환경 변수 사용)
* **운영(리얼) 서버:** 포트 `3000` 사용 (도커 배포 시)

## 세계관 및 캐릭터 설정 (Worldview & Characters)

이 앱은 가족 간의 긍정적인 행동 강화와 보상을 위한 서비스입니다. 다음 캐릭터 설정을 기억하고 UI 로직 및 대화 톤앤매너에 반영하세요.

*   **제리(Jerry):** 쥐(🐭) 캐릭터입니다. 사용자가 '할 일'을 통해 콩을 획득하거나, 모은 콩을 사용할 때 그 양과 가치를 **가장 공정하게 판정하는 심판(판사)** 역할을 담당합니다.
